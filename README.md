# SaferLinks

SaferLinks is a middleman website that inspect external links for further redirections.

With SaferLinks, users will be warned if the content referenced in the external URL requires a mobile app, a platform subscription, or to log in into a specific account. We have a growing list of database that is stored locally, and you can optionally extend with third-party API services like CrossRef to gather more information for the external link.

SaferLinks will also warn users if certain analytics or advertising cookies are being used within the target external websites, making sure that your users consent to have their data collected by the third parties in their respective platforms.
